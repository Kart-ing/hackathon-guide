# HackGuide (Midnight Hacker Theme)

A curated hub for hackathon discovery, ideation, development tools, AI/ML, and winning strategies. Built with Next.js App Router, Tailwind, Framer Motion, and React Hot Toast.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

- Push to GitHub and import the repo on Vercel
- Framework preset: Next.js
- Root directory: `hackathon-guide`
- Build command: `next build` (auto)
- Output: `.next` (auto)

Analytics are enabled via `@vercel/analytics` in `app/layout.tsx`.
